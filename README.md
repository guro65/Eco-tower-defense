# Eco-tower-defense

Tower defense contendo personagens e cenários inspirados em um Rpg antigo feito por amigos. O jogo é desenvolvido em C#.
Equipe pequena de 3 pessoas, onde cada um tem uma função diferente, como programador, artista e designer. O jogo é focado em ser divertido e desafiador, com gráficos simples em 2d. O objetivo do jogo é defender sua base contra ondas de inimigos, utilizando diferentes tipos de personagens e estratégias para vencer.
